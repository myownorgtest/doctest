    Base class for all REST services settings.

    Derived classes must implement the following members: 1- $isInitialized: A static property that indicates whether the class's static members have been initialized. 2- init(): A protected static method that initializes static members. 3- $validSettingKeys: A static property that contains valid setting keys for this service. 4- createFromConnectionString($connectionString): A public static function that takes a connection string and returns the created settings object.
