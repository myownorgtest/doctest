About This Documentation¶

This documentation is generated using the Sphinx documentation generator. The source files for the documentation are located in the doc/ directory of the PyDocumentDB distribution. To generate the docs locally run the following command from the root directory of the PyDocumentDB source:

$ python setup.py install
$ cd doc
$ sphinx-apidoc -e -o .\api ..\pydocumentdb
$ make.bat html
