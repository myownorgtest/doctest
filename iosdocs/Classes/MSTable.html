Overview

The MSTable class represents a table of a Microsoft Azure Mobile Service.

Items can be inserted, updated, deleted and read from the table. The table can also be queried to retrieve an array of items that meet the given query conditions. All table operations result in a request to the Microsoft Azure Mobile Service to perform the given operation.
