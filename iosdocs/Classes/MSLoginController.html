Overview

The MSLoginController class provides a UIViewController that can be presented to allow an end user to authenticate with a Microsoft Azure Mobile Service.

toolbar

The UIToolbar associated with the MSLoginController.

@property UIToolbar *toolbar
Discussion

The toolbar includes a cancel button and an activity indicator. The visibility and placement of the toolbar can be configured using the showToolbar and toolbarPosition properties respectively.
