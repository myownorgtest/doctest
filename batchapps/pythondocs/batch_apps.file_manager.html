FileManager¶

class batchapps.file_manager.FileManager(credentials, cfg=None)[source]¶

This is the only class that a user should need to import to access all user data and asset files manipulation. Contains only general functionality for the creation of UserFile and FileCollection objects.

Args:
  • credentials (Credentials): The Users authentication.
Kwargs:
  • cfg (Configuration): Logging and application configuration. Unless set this is None, in which case a default configuration will be used.

files_from_dir(top_dir, recursive=False, pattern='*')[source]¶

Create a file set from contents of a directory.

Args:
  • top_dir (str): The full path to the directory, the contents of which will be added to the set.
Kwargs:
  • recursive (bool): Whether the set will include the contents of subdirectories. Default is False
  • pattern (str): The pattern to apply to filter director contents. Default: "*", i.e. all files included.
Returns:
  • FileCollection The new FileCollection containing an UserFile object for each non-filtered file in the specified directory.
Raises:
  • NotADirectoryError if the supplied directory does not exist.
