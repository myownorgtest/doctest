Class: NamespaceOperations

NamespaceOperations

The Service Bus Management API includes operations for managing Service Bus namespaces. NOTE: An instance of this class is automatically created for an instance of the [ServiceBusManagementClient] ServiceBusManagementClient~ServiceBusManagementClient. See [namespaces] ServiceBusManagementClient~ServiceBusManagementClient#namespaces. Initializes a new instance of the NamespaceOperations class.
