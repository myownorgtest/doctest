Class: DatabaseCopyOperations

DatabaseCopyOperations

The SQL Database Management API includes operations for managing SQL Database Copies for a subscription. NOTE: An instance of this class is automatically created for an instance of the [SqlManagementClient] SqlManagementClient~SqlManagementClient. See [databaseCopies] SqlManagementClient~SqlManagementClient#databaseCopies. Initializes a new instance of the DatabaseCopyOperations class.
