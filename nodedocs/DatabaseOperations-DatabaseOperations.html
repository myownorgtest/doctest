Class: DatabaseOperations

DatabaseOperations

The SQL Database Management API includes operations for managing SQL Databases for a subscription. NOTE: An instance of this class is automatically created for an instance of the [SqlManagementClient] SqlManagementClient~SqlManagementClient. See [databases] SqlManagementClient~SqlManagementClient#databases. Initializes a new instance of the DatabaseOperations class.
