Class: ServerOperations

ServerOperations

The SQL Database Management API includes operations for managing SQL Database servers for a subscription. (see http://msdn.microsoft.com/en-us/library/windowsazure/gg715271.aspx for more information) NOTE: An instance of this class is automatically created for an instance of the [SqlManagementClient] SqlManagementClient~SqlManagementClient. See [servers] SqlManagementClient~SqlManagementClient#servers. Initializes a new instance of the ServerOperations class.
