Class: ApnsService

ApnsService

The ApnsService class is used to send notifications using the Apple Push Notification Service (APNS). This service is used to communicate with iOS device.

new ApnsService(notificationHubService)

Creates a new ApnsService object using the specified NotificationHubService object.

Note: An instance of this object is created automatically when a NotificationHubService object is created. See NotificationHubService#apns.

Parameters:
Name Type Description
notificationHubService NotificationHubService

The notification hub service.
